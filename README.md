Notebooks from my Masters in High Performance Computing programme covering quantum circuit fundamentals, standard algorithms, and QUBO-based optimisation.

## Notebooks

**Qiskit & Cirq**
- `qiskit_entanglement.ipynb` — Qiskit fundamentals, quantum circuits, entanglement, Bell states
- `quantum_algorithms.ipynb` — Deutsch-Jozsa, Bernstein-Vazirani, quantum phase estimation
- `grovers_cirq.ipynb` — Grover's search algorithm in Cirq

**QUBO & Optimisation (PyQubo)**
- `pyqubo_intro.ipynb` — Getting started with PyQubo
- `game_of_switches.ipynb` — QUBO formulation of a combinatorial switch problem
- `qubo_antenna_placement.ipynb` — Antenna placement as a QUBO problem
- `qubo_antenna_equal_constraint.ipynb` — Equality constraints
- `qubo_antenna_linear_constraint.ipynb` — Linear constraints
- `qubo_antenna_leq_constraint.ipynb` — Less-than-or-equal constraints
- `qubo_antenna_higher_order.ipynb` — Higher-order QUBO terms
- `budget_optimisation.ipynb` — Budget allocation as a QUBO optimisation problem
